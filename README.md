## 🧠 О себе

Занимаюсь разработкой решений в области машинного обучения и компьютерного зрения. В основном работаю с задачами анализа изображений: от базовой обработки до построения простых пайплайнов, где изображение проходит несколько этапов — предобработка, выделение нужной области и дальнейший анализ.

Интересуюсь практическим применением ML и CV — не просто моделями, а системами, которые можно использовать в реальных задачах.
